Androvega Raspberry Pi Setup Guide

Raspberry Pi 4 and Raspberry Pi 5 installation, AnyDesk and autostart instructions.

Security warning: Never publish AnyDesk IDs, unattended-access passwords or customer credentials on a public website. Replace the placeholders locally and change all default passwords after installation.

Raspberry Pi 4

Prerequisites

  • Device version: Raspberry Pi 4
  • Operating system: Raspberry Pi OS Legacy 32-bit

Reinstall Raspberry Pi OS

  1. Reboot the Raspberry Pi while holding the Shift key.
  2. Select Raspberry Pi OS Legacy 32-bit.
  3. Correctly configure the country, keyboard and timezone.
  4. Set the username to pi.
  5. Create a secure password and save it in the approved password manager.

Install AnyDesk on Raspberry Pi 4

  1. Download the AnyDesk Raspberry Pi armhf.deb version.
  2. Install the downloaded package.
  3. Run anydesk from the terminal.
  4. If AnyDesk reports a GLES or EGL library error, run the commands below. 
sudo apt update
  sudo apt install libgles-dev libegl-dev

  sudo ln -s /usr/lib/arm-linux-gnueabihf/libGLESv2.so /usr/lib/libbrcmGLESv2.so
  sudo ln -s /usr/lib/arm-linux-gnueabihf/libEGL.so /usr/lib/libbrcmEGL.so

Enable unattended access

  1. Open AnyDesk on the Raspberry Pi.
  2. Open Settings → Security.
  3. Enable Unattended Access.
  4. Set a unique password.
AnyDesk password: [DO NOT PUBLISH — SET LOCALLY]

HDMI Configuration

For older Raspberry Pi OS versions:

sudo nano /boot/config.txt

For Raspberry Pi OS Bookworm:

sudo nano /boot/firmware/config.txt

Add or update the following values:

disable_overscan=0
  hdmi_force_hotplug=1
  hdmi_group=2
  hdmi_drive=2
  config_hdmi_boost=4

  Save:            Ctrl + O, then Enter
  Exit:            Ctrl + X
  Exit after save: Ctrl + X

Install an Androvega Application

  1. Get the device's AnyDesk ID from the approved Excel sheet.
  2. Connect to the correct Customer, Boxing, Kiosk, Kitchen, Map, Pizza Maker or Cashier device.
  3. Select the correct download URL for the operating system architecture.
  4. Download the AppImage.
  5. Make the downloaded AppImage executable.
cd /home/pi/Downloads
  chmod +x application-name.AppImage

Example:

chmod +x /home/pi/Downloads/androvega-pos-setup-0.4.23.AppImage

Raspberry Pi 4 Autostart

Open the LXDE autostart file:

sudo nano /etc/xdg/lxsession/LXDE-pi/autostart

Add the application path on a new line:

@/home/pi/Downloads/androvega-pos-setup-0.4.23.AppImage

Application configuration

  • Open Developer Tools with Ctrl + Shift + I.
  • Initial application login password: 123456.
  • Change the default password after setup.
  • Server URL: http://192.168.1.190
  • Socket protocol: ws://, not wss://.

Create a desktop shortcut

  1. Open Preferences.
  2. Open Main Menu Editor.
  3. Select New Item.
  4. Enter the application name and AppImage path.

Raspberry Pi 5

Enable desktop autologin

sudo raspi-config

Select: System Options → Boot / Auto Login → Desktop Autologin

Create an application autostart file

mkdir -p ~/.config/autostart
  nano ~/.config/autostart/boxing.desktop

Paste the following configuration:

[Desktop Entry]
  Type=Application
  Name=Boxing Screen
  Exec=/home/pi/Downloads/androvega-boxing-setup-0.4.21.AppImage
  Terminal=false
  X-GNOME-Autostart-enabled=true

Make both files executable:

chmod +x ~/.config/autostart/boxing.desktop
  chmod +x /home/pi/Downloads/androvega-boxing-setup-0.4.21.AppImage

Start AnyDesk automatically

mkdir -p ~/.config/autostart
  cp /usr/share/applications/anydesk.desktop ~/.config/autostart/

Restart the device:

sudo reboot

Raspberry Pi 5 uses a 64-bit operating system. Download the arm64 application whenever an ARM64 version is available.
